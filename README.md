## Getting Started

### Using Claude Skills

```bash
# Install a skill
cp -r claude-skills/book-to-visual-companion ~/.claude/skills/

# Use in Claude Code
/book-to-visual-companion
```

See [claude-skills/README.md](./claude-skills/README.md) for details.
